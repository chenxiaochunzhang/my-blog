什么是 Markdown？

Markdown 是一种轻量级标记语言，让你可以用简单的语法写出格式优美的文章。

为什么用 Markdown？

  • 简单易学：几分钟就能上手
  • 专注内容：不用纠结排版
  • 随处可用：GitHub、知乎、简书都支持

基本语法

标题

# 一级标题
## 二级标题
### 三级标题

列表

  • 项目1
  • 项目2
  • 项目3

代码

行内代码：print("Hello")

代码块：

def hello():
    print("Hello, World!")

开始使用

现在你可以用 Markdown 写博客了！只需要：

  1. 在 content/posts/ 目录创建 .md 文件
  2. 运行 python build.py 生成 HTML
  3. 上传到 GitHub
